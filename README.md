# Portfólio de Projetos em Análise de Dados

Este repositório reúne três projetos desenvolvidos com foco em análise de dados utilizando Python (Pandas, Matplotlib) e conceitos de SQL.

Os dados utilizados são fictícios e foram criados para fins educacionais e demonstração técnica.

---

# Projeto 1 — Análise de Rentabilidade

## Objetivo
Avaliar margem de lucro, identificar produtos mais rentáveis e analisar eficiência financeira.

## Principais Análises
- Receita total por produto
- Lucro absoluto
- Margem de lucro (%)
- Rentabilidade por categoria

## Principais Insights
- A **Calça Jeans** apresentou o maior lucro absoluto no período analisado.
- A margem média da empresa foi de aproximadamente **45,7%**, indicando boa eficiência operacional.
- A categoria **Calçados** apresentou maior rentabilidade proporcional.
- Produtos com menor margem podem ser avaliados para renegociação de custos ou ajustes estratégicos de preço.

# Projeto 2 — Performance Comercial

## Objetivo
Analisar o desempenho da equipe comercial comparando meta e resultado por vendedor e região.

## Principais Análises
- Meta vs Realizado
- Percentual de atingimento de meta
- Ranking de vendedores
- Performance por região

## Principais Insights
- O vendedor com maior faturamento foi **Carlos Mendes**, demonstrando alta performance comercial.
- A região **Sudeste** apresentou o maior volume total de vendas.
- A média de atingimento de meta da equipe foi superior a 100% em determinados períodos.
- Vendedores abaixo da meta podem demandar ações de treinamento ou revisão estratégica.

# Projeto 3 — Análise de Comportamento de Clientes

## Objetivo
Identificar padrões de consumo e perfil de clientes em um e-commerce fictício.

## Principais Análises
- Receita por categoria
- Ticket médio por gênero
- Faixa etária com maior consumo
- Receita por cidade

## Principais Insights
- A categoria **Moda Feminina** apresentou maior faturamento total.
- O público feminino demonstrou ticket médio superior.
- A faixa etária de 25–35 anos concentra maior volume de compras.
- Capitais como São Paulo e Rio de Janeiro apresentam maior valor acumulado de vendas.
- Existe potencial para estratégias de fidelização focadas nos clientes com maior ticket médio.
